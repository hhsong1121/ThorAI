# Training Data Layout

This repository does **not** include proprietary datasets, trained weights, or copyrighted guideline PDFs. Use the layouts below after obtaining data under each dataset's license.

## VinDr-CXR (object detection)

Used by:

- `utils_data/prepare_vindr.py`
- `model_monai/train_vindr_detection.py`
- `model_monai/eval_vindr.py`

Expected layout:

```text
research_and_training/
  utils_data/
    train.csv                 # raw VinDr annotation CSV (not in repo)
    master_vindr_labels.csv   # generated by prepare_vindr.py
  model_monai/
    vinbigdata/
      train/
        {image_id}.png
    checkpoints/
      vindr_det_epoch_*.pth
```

Environment variables: see `.env.example` (`VINDR_CSV_PATH`, `VINDR_IMG_DIR`, `VINDR_CHECKPOINT_DIR`).

Source: [VinDr-CXR on PhysioNet](https://physionet.org/content/vindr-cxr/1.0.0/) — credential required.

Pipeline:

1. Download `train.csv` and images under license.
2. Run `python utils_data/prepare_vindr.py` to build `master_vindr_labels.csv`.
3. Run `python model_monai/train_vindr_detection.py`.
4. Copy the best checkpoint to `backend_api/weights/vindr_det_epoch_10.pth` for production inference.

## NIH Chest X-ray + CheXmask (segmentation)

Used by: `model_monai/CXR_ft.py`

Expected layout:

```text
/path/to/NIH_CXR/images/
  00000001_000.png
  ...
/path/to/CheXmask/Preprocessed/
  00000001_000.png
  ...
```

Environment variables: `NIH_CXR_DIR`, `CHEXMASK_DIR` (or `NIH_CXR_BASE`, `CHEXMASK_BASE`).

- NIH: [ChestX-ray14](https://nihcc.app.box.com/v/ChestXray-NIHCC)
- CheXmask: see the paper/repository referenced in your local notes

## Synthetic SLM fine-tuning data

Used by: `model_slm/make_dataset.py`, `model_slm/convert_to_mlx.py`

No external dataset required. Generated files (`train.jsonl`, `mlx_data/`) are gitignored.

## Guideline PDFs (RAG)

Used by: `rebuild_db.py`

Place licensed PDFs in `research_and_training/data/` (gitignored), then run:

```bash
cd research_and_training
python rebuild_db.py
```

Output: `database/qdrant_db/` (gitignored).

For the public portfolio demo, the app uses paraphrased snippets in `backend_api/demo_assets.py` instead of PDF-derived embeddings.

## Inference weights (production only)

| File | Purpose |
|------|---------|
| `backend_api/weights/vindr_det_epoch_10.pth` | VinDr Faster R-CNN detector |
| `backend_api/adapters/*.safetensors` | MLX LoRA adapters |

None of these are included in the public repository.
